# PROJECT CONTEXT — BaanFlow

## Product Name
**BaanFlow**

## Business Type
Vertical SaaS / web app for Thai prefab home, knockdown home, and modular building companies.

## One-line Positioning
ระบบจัดการแบบบ้าน ใบเสนอราคา และงานติดตั้งบ้านน็อคดาวน์ ตั้งแต่ลูกค้าเลือกแบบจนถึงส่งมอบงาน

## Target Audience
- บริษัทบ้านน็อคดาวน์ / บ้านสำเร็จรูป / modular home
- บริษัทรับสร้างบ้านขนาดเล็ก-กลางที่ขายแบบมาตรฐาน + custom option
- ทีมขายและทีมโครงการที่ยังใช้ LINE, Google Drive, Excel, และเอกสารแยกกัน

## Primary Pain Points
- แบบบ้านและไฟล์กระจัดกระจายใน LINE/Drive
- เซลส์ตีราคา option หรือค่าขนส่งผิด
- ใบเสนอราคามีหลาย version แล้วตามไม่ทันว่า version ไหนอนุมัติแล้ว
- ลูกค้าถามสถานะงานบ่อย เพราะไม่มี portal ดูความคืบหน้า
- งวดเงิน/มัดจำ/ก่อนผลิต/ก่อนส่ง/ส่งมอบ หลุดการติดตาม
- ไม่รู้กำไรจริงต่อหลัง เพราะต้นทุนวัสดุ ค่าแรง ขนส่ง และค่าแก้งานไม่ถูกรวมในที่เดียว

## What This Project Is NOT
- ไม่ใช่ ERP/accounting SaaS เต็มรูปแบบ
- ไม่แข่งตรงกับ FlowAccount, PEAK, SMEMOVE ในบัญชี/VAT/payroll
- ไม่ทำ GL, ปิดงบ, e-Tax submission, payroll compliance เต็มใน MVP
- ไม่ใช่ marketplace ขายบ้านให้ end-customer ใน v1
