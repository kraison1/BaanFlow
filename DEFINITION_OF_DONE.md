# DEFINITION OF DONE — BaanFlow MVP Definition

MVP definition is done when:
- ICP is selected and documented.
- Feature scope is split into v1, v1.1, and out-of-scope.
- Competitor matrix is documented.
- Data model draft is documented.
- Next.js SEO page map is documented.
- User flow from upload model → quotation → project → customer portal is documented.
- Risks and pricing hypothesis are documented.
